# Web Animations

## Ne zaman
Smooth UX, brand polish, attention.

## Stack
Framer Motion + GSAP + ScrollTrigger

## Süreç
1. Motion library seç (Framer simple, GSAP advanced)
2. Page transitions
3. Scroll-triggered animations
4. Hover effects
5. Micro-interactions (button, form)
6. Performance (will-change, transform only)
7. Reduced motion respect

## Çıktı standardı
- 60fps consistent
- Reduced motion alternatif
- Mobile performant
- prefers-reduced-motion

## Yaygın hatalar

- Mobile battery drain
