# Internationalization

## Ne zaman
Multi-language site.

## Stack
next-intl + JSON translations

## Süreç
1. Locale detection (path / cookie / header)
2. Translation files (JSON per language)
3. Server + client translations
4. Date/number formatting
5. Pluralization
6. RTL support (Arabic, Hebrew)
7. Language switcher

## Çıktı standardı
- All strings extracted
- Pluralization handled
- RTL CSS works
- SEO per locale (hreflang)

## Yaygın hatalar
- Hard-coded strings missed
- Plural rules wrong
- Date format locale-aware değil
