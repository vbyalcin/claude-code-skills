# Fitness App

## Ne zaman
Personal trainer, gym, fitness coach.

## Stack
Next.js / React Native + Supabase

## Süreç
1. Exercise library (video + animation)
2. Workout program (haftalık)
3. Set/rep tracking
4. Progress charts (PR'lar)
5. Body measurements
6. Nutrition tracking
7. Coach communication

## Çıktı standardı
- Mobile-first (gym'de telefon)
- Offline mode (signal yok)
- Wearable sync (Apple Health)
- Video stream optimize

## Yaygın hatalar
- Battery drain (background tracking)
- Apple Health permission flow
- Workout pause/resume
