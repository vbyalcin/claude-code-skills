# Form Builder

## Ne zaman
Müşteri kendi form'larını oluştursun.

## Stack
Next.js + dnd-kit + Supabase

## Süreç
1. Field types (text, email, select, multi-choice, file)
2. Drag-drop builder UI
3. Logic (conditional fields)
4. Theming (color, font)
5. Public form URL
6. Response collection
7. Webhook + Sheets export

## Çıktı standardı
- Mobile-friendly
- Conditional logic
- Multi-step support
- Spam protection (honeypot)

## Yaygın hatalar
- Conditional logic karmaşık
- File upload virus scan yok
- Response retention policy yok
