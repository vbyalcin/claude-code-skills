# Markdown Editor

## Ne zaman
Blog yazma, doc editing, technical writing.

## Stack
Tiptap / Lexical / CodeMirror + Markdown

## Süreç
1. Editor + preview pane
2. Toolbar (format buttons)
3. Image paste/drag-drop upload
4. Code blocks (syntax highlight)
5. Slash commands (/heading, /code)
6. Auto-save
7. Export (Markdown, HTML, PDF)

## Çıktı standardı
- WYSIWYG ya da Markdown raw toggle
- Image upload to S3/R2
- Tables, footnotes, math
- Mobile keyboard friendly

## Yaygın hatalar
- Image upload sync (sayfayı bloklar)
- Table editing UX zayıf
- Mobile selection problemleri
