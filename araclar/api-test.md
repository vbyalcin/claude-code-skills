# API Testing Tool

## Ne zaman
Dev/QA için API test yardımcısı.

## Stack
Next.js + react-codemirror

## Süreç
1. Request builder (method, URL, headers, body)
2. Response viewer (JSON, headers, status)
3. Collections (organize)
4. Environments (variables)
5. Auth helpers (Bearer, OAuth)
6. Tests (assertions)
7. Run history

## Çıktı standardı
- Save + share collections
- Environment variable substitution
- JSON syntax highlighting
- Performance metrics

## Yaygın hatalar
- Secrets save plaintext
- CORS issues (use proxy)
- History storage limits
